
# 使用说明

## 1、下载安装Edge浏览器

保证你的电脑上有Edge浏览器，因为这个脚本是运行在Edge浏览器的。

## 2、运行exe

运行可执行exe程序之后，选择一个规则文件，后续将自动运行即可

![alt text](image.png)

## 3、修改yaml配置文件

如果需要修改填写规则，可以新建或者修改`rules`文件夹下的yaml文件。

下图中①表示问题类型（单选或者是矩阵单选），②表示各个问题选项选取的概率

![alt text](image-1.png)

## 注意事项

- 运行后，将会自动打开Edge浏览器，并开始填写问卷。（不会自动停止，只有出错或者按下ESC键时才会退出程序）
- 程序运行期间，请不要操作电脑，以免影响自动填写过程。（影响自动点击智能验证，可以在去吃饭的时候，让它自己跑，这样不影响学习工作）
- 确保网络畅通，以免影响网页加载和提交。（超时出错也会中止程序）
- 在命令行窗口会有括号括起来的一个数字，那是填写问卷的次数，当觉得填的份数差不多的时候，可以先激活命令行窗口，再按下ESC键结束程序。结束程序之后，程序会自动关闭浏览器，并在命令行中显示"程序结束"。
