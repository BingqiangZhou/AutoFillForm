

# 自动问卷填写需求

## 一、主要需求

1. 填入问卷链接，分析题目，展现题目到txt文件中
2. 在分析得到的txt文件中，可以设定每个问题选择概率填写问卷
3. 设定结束后可以导出函数到txt文件，文件中是问题调用的函数，参数设置包括问题选项的概率
4. 在导出函数的txt文件中设定问题与问题之间的相关性，当一个问题选择一个选项之后，设定另一个问题选项填写的概率
5. 数据回填：根据excel文档设置的问题选项填写问卷，excel每一行数据为一份问卷填写的每个选项对应的编号，每一列对应一个问题


## 二、其他需求

1. 填入问卷链接：设置问卷填写次数，提交时间（小时、分钟、秒、运行倒计时【待分析】）
2. 设置代理ip、包括api链接、ip端口、查看当前ip，查看ip归属地（保存数据、清理数据【待分析】）
3. 浏览器配置，包括是否打开浏览器，问卷来源（保存数据、一键暂停所有、运行1、运行2...【待分析】）
4. 问题作答设置，（设置回填、清理所有数据、刷新日志【待分析】）

## 三、隐性需求

1. 所有类型的题型都能填写【待分析】
2. 每一份填写的出来的问卷ip都不一样【待分析】
3. 支持问卷星、腾讯问卷、见数问卷、qualtrics问卷
