
# 使用说明

## 1、安装Python环境以及相关依赖包

确保你的电脑上已经配置好了Python环境，并安装好下面这些依赖库。

```code
pip install selenium pyautogui pywin32 webdriver_manager
```

## 2、下载安装Edge浏览器

保证你的电脑上有Edge浏览器，因为这个脚本是运行在Edge浏览器的。

## 3、运行Python脚本

1. 在命令行中进入脚本所在的目录。
2. 运行脚本，可以使用命令`python v6.py`

## 注意事项

- 运行脚本后，将会自动打开Edge浏览器，并开始填写问卷。（不会自动停止，只有出错或者按下ESC键时才会退出程序）
- 程序运行期间，请不要操作电脑，以免影响自动填写过程。（影响自动点击智能验证，可以在去吃饭的时候，让它自己跑，这样不影响学习工作）
- 确保网络畅通，以免影响网页加载和提交。（超时出错也会中止程序）
- 在命令行窗口会有括号括起来的一个数字，那是填写问卷的次数，当觉得填的份数差不多的时候，可以先激活命令行窗口，再按下ESC键结束程序。结束程序之后，程序会自动关闭浏览器，并在命令行中显示"程序结束"。
