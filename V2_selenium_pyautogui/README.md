
# 使用说明

## 1、下载安装Edge浏览器

保证你的电脑上有Edge浏览器，因为问卷是通过打开Edge浏览器自动填写的。

## 2、修改yaml配置文件

根据自己的要求，修改自动填写规则，可以新建或者修改`rules`文件夹下的yaml文件。

下图中①表示问题类型（单选、多选、矩阵单选），②表示各个问题选项选取的概率，其中，
- 单选`radio_selection`后面接的列表是单个选项被选择的权重，概率计算方式为`x/sum([])`，例如[20,30,50]，则第一个选项被选择得概率为`20%=(20/sum(20,30,50))`。
- 多选`radio_selection`后面接的列表是每一个选项被选择得概率，概率计算方式为`x/100`，例如[50,20,50]，则第一个选项被选择得概率为`50%=(50/100)`，第二个选项被选择得概率为`20%=(20/100)`。
- 矩阵单选`matrix_radio_selection`后面接的列表是与单选`radio_selection`，不同的是矩阵单选后面跟了多个单选的概率。

**注意**：所有概率值都应该为整数。

## 3、运行exe

运行可执行exe程序之后，选择一个规则文件，后续将自动运行即可。

![alt text](image.png)

## 注意事项

- 运行后，将会自动打开Edge浏览器，并开始填写问卷。（不会自动停止，只有出错或者按下ESC键时才会退出程序）
- 程序运行期间，请不要操作电脑，以免影响自动填写过程。（影响自动点击智能验证，可以在去吃饭的时候，让它自己跑，这样不影响学习工作）
- 确保网络畅通，以免影响网页加载和提交。（超时出错也会中止程序）
- 在命令行窗口会有括号括起来的一个数字，那是填写问卷的次数，当觉得填的份数差不多的时候，可以先激活命令行窗口，再按下ESC键结束程序。结束程序之后，程序会自动关闭浏览器，并在命令行中显示"程序结束"。
